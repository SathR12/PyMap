# PyMap

Tile Map and Auth GUI using tkinter and tkintermapview 

## How to run

Git clone this repo, edit the path strings, and run the authentication.py file 

## Dependencies

- Tkinter
- TkinterMapView
- Json 
- Random
- Sys
- Yagmail
